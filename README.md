    - 집합의 데이터가 정렬되어 있지 않다면 이 검색 외에 특별한 방법 없음

- 이진탐색

    - 순차 검색에 비해 월등히 효율적이라 데이터가 몇 천만 개 이상이어도 빠르게 찾아낼 수 있음
